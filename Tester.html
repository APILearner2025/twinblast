Sample Queries for Draft Information

Click on any sample query to load it into the query editor. These are common patterns for discovering draft-related data:

Schema Introspection

{
  __schema {
    queryType {
      fields {
        name
        description
      }
    }
  }
}

Find Match Types

{
  __schema {
    types {
      name
      description
    }
  }
}

Query Matches (Example)

{
  matches {
    id
    status
    startTime
    teams {
      name
      players {
        name
        hero
      }
    }
    draft {
      picks
      bans
      order
    }
  }
}

Query Tournaments (Example)

{
  tournaments {
    id
    name
    matches {
      id
      teams {
        name
        draft {
          heroSelections
          pickOrder
        }
      }
    }
  }
}

Hero/Champion Data (Example)

{
  heroes {
    id
    name
    role
    pickRate
    winRate
  }
}

Draft Predictions (Example)

{
  predictions {
    matchId
    draftPredictions {
      team
      predictedPicks
      confidence
    }
  }
}

Instructions

  1. Test Connection: First, test if the API endpoint is accessible
  2. Schema Discovery: Use the introspection query to discover available data types and fields
  3. Explore Data: Based on the schema, craft specific queries for draft information
  4. Common Draft Fields: Look for fields like picks, bans, draft, heroSelection, pickOrder, champions

Note: This tool helps you explore the API. You may need authentication headers or specific parameters that aren't shown here. Check the API documentation for authentication requirements.
