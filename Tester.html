Instructions for Timeline Data Discovery

  1. Test Connection: First, verify the API endpoint is accessible
  2. Schema Discovery: Use introspection to find timeline-related types and fields
  3. Find Match Query: Discover how to query matches by ID
  4. Timeline Fields: Look for fields like:
    • timeline, events, actions
    • purchases, items, itemEvents
    • damage, damageEvents, combatLog
    • abilities, spells, abilityEvents
    • timestamp, gameTime, matchTime
  5. Test with Sample ID: Try different match ID formats to find the correct structure

  • Timeline data is usually the most resource-intensive to generate

  • Check if there are different detail levels (summary vs full timeline)

Note: Detailed match timelines require significant data processing. The API might:

  • Require special permissions or authentication
  • Have rate limiting for timeline queries
  • Only provide timeline data for recent matches
  • Separate timeline data into multiple queries
